# StudyStride - 学习打卡应用

StudyStride 是一个简洁高效的学习打卡应用，帮助用户记录学习时间、追踪学习进度，并在适当的时候提醒休息。

## 功能特性

- ⏱️ **学习计时**：开始/暂停/停止计时器，实时显示学习时长
- 📝 **学习内容记录**：可选择性记录当天的学习内容
- 📅 **每日打卡系统**：自动记录每日学习次数和总时长
- 🔔 **休息提醒**：学习时长达到设定时间后发送休息提醒
- 📊 **历史记录**：按日期查看历史学习记录
- 💾 **数据持久化**：本地存储学习数据，支持数据导出
- 🎨 **响应式设计**：适配不同设备屏幕

## 技术栈

- Vue 3 + TypeScript
- Pinia（状态管理）
- Vue Router（路由）
- Vite（构建工具）
- LocalStorage（数据存储）

## 使用指南

### 开始学习
1. 在首页点击「开始学习」按钮启动计时器
2. 可选：在文本框中记录学习内容
3. 点击「暂停」按钮可暂停计时，点击「继续」按钮恢复计时
4. 点击「结束」按钮完成学习 session

### 查看历史记录
- 在导航栏点击「历史记录」查看所有学习记录
- 可选择特定日期查看当天的学习记录

### 配置设置
- 在导航栏点击「设置」进入设置页面
- 可调整休息提醒的间隔时间
- 可请求通知权限以接收休息提醒
- 可导出或清除学习数据
